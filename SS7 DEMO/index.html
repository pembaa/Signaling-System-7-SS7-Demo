Why SS7 matters for rural Nepal

SS7 is the set of signaling rules that help networks set up your calls and SMS. It was not built for modern security — attackers can misuse it to intercept SMS (OTP), redirect calls, or track phones. This tool safely demonstrates those risks and gives practical tips.
